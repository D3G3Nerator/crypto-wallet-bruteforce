
# Wallet Generator w/ Balance

Generates random wallets using **ethers.Wallet** and checks for balance in them using a node. (wss)
## Installation

Be sure to have NPM, Yarn installed.

```bash
  yarn add ethers
  yarn add web3
  node bot.js
```
